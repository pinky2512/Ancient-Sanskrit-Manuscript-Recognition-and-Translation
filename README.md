# 🕉️ Ancient Sanskrit Manuscripts Recognition and Translation








An AI-powered system for digitizing, recognizing, and translating ancient Sanskrit manuscripts using Optical Character Recognition (OCR) and automated translation tools.

The goal of this project is to preserve cultural heritage and make historical Sanskrit texts accessible through modern machine learning tools.

## 📖 Introduction

Ancient Sanskrit manuscripts contain vast knowledge in fields such as philosophy, medicine, mathematics, and literature. Unfortunately, many of these manuscripts remain inaccessible due to:

Physical degradation of historical documents

Limited digitization efforts

Difficulty in reading ancient scripts

This project builds an AI-driven pipeline to extract text from manuscript images and translate it into modern languages. By combining OCR and translation technologies, the system helps transform ancient handwritten or printed manuscripts into readable digital text.

## 🔎 Overview

The system processes images of Sanskrit manuscripts through multiple stages:

1. Image preprocessing to improve clarity

2. OCR-based text extraction

3. Post-processing of extracted Sanskrit text

4. Automated translation to modern languages

This pipeline allows researchers, historians, and linguists to analyze and interpret ancient manuscripts more efficiently.

## ⚙️ Features

📜 Extract Sanskrit text from manuscript images using OCR

🧠 Automated translation of Sanskrit text into modern languages

🖼 Image preprocessing for improved OCR accuracy

🔄 End-to-end processing pipeline from image → translated text

📂 Support for processing multiple manuscript images
